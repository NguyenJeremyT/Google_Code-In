How I made this Website

Step 1: Accepting the Task on Google Code-In

I stumbled upon the task on Google Code-In. I accepted it.

Step 2: Figure out the basic structure of a HTML page

I looked through the w3 schools tutorial on HTML and figured out the general structure of an HTML page. I ended up using Notepad++ for my choice of editor.

Step 3: Figure out CSS

After that, I looked through the w3 schools tutorial on CSS and figure out CSS.

Step 4: Use BootStrap

I found out about BootStrap and figured it would look nicer and be more usable than my own pure CSS. I mostly used the navbar and its container features.

Step 5: The Rest of the Website

This is when a made a little small changes. I made Roboto the font for the website using Google Fonts. Since I had to put an image, I put this one from Unsplash (so I wouldn't be breaking copyright). After that, I had to figure out how to put text on an image, which took [1], [2], [3] links because apparently centering text is extremely difficult. The rest of the website is made up of purely h1 and p tags since after all the time I spent figuring how to center text on an image left me with only 1 day left of the task. :/

Step 6: Add a Favicon and Meta Tags

I had no idea what a favicon was so I looked it up on w3schools. I ended up just using the favicon from fortnite.com because I had no creative ideas for a favicon. I found out what meta tags were on w3schools too and added them in.
